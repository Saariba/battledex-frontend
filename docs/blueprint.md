# **App Name**: Rap Battle Lines

## Core Features:

- Semantic Search: Enable semantic search using the API to find relevant punchlines based on meaning, not just keywords.
- Keyword Search: Implement keyword search to find punchlines that match the query term.
- Result Card Display: Display search results in cards with the core punchline emphasized and context lines available to view.
- Video Modal: Create a modal that plays the YouTube video at the timestamp of the selected punchline.
- Battle Listing: Display list of available battles from the API with title, league and link.
- Rapper Listing: Display list of available rappers from the API for adding a rapper filter to search.

## Style Guidelines:

- Primary color: Saturated red (#EF4444) to highlight key elements and scores.
- Background color: Near black (#0A0A0A) for a dark, gritty feel.
- Accent color: Soft blue (#3B82F6) for links and buttons.
- Body font: 'Inter' (sans-serif) for readability, with bold variations for headings and the core punchline.
- Code font: 'Source Code Pro' (monospace) for timestamps, line numbers.
- High-contrast dark mode for readability.
- Result Cards for punchline and video context.
